
Instrucciones para ejecutar el programa:
 -Debido a que es un proyecto de Netbeans se puede abrir con este IDE y ejecutarse.

El programa crea una gráfica aleatoria y crea permutaciones de vértices aleatorias.
Para verificar que sí funciona se puede probar con un ejemplo sencillo (un pentágono)
cambiando la línea 29 (ejAleatorio();) de la clase Principal por "ejFijo();".

Para aumentar la velocidad (los fps) se puede cambiar el argumento de frameRate (línea 27).

Nota: Cuando se muestra en pantalla "No es hamiltoniano" significa que el certificado elegido
hasta el momento no es un ciclo hamiltoniano, no significa que la gráfica no tenga un 
ciclo hamiltoniano.
