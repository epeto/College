
Primero se debe ejecutar la clase "GenLec" para generar los números, con el comando:

java GenLec <tam> <archivo>

Donde <tam> es la cantidad de números que se van a generar y <archivo> es el nombre del archivo donde se escribirán los números. Luego, se observa algún número en esta secuencia y se elige uno. Una vez que se eligió el número(digamos que es n) se ejecuta la clase Busquedas con el comando:

java Busquedas <archivo> n <algoritmo de búsqueda>

Donde

<algoritmo de búsqueda> ::= secuencial
                            binaria
                            exponencial
                            interpolacion

y <archivo> es el archivo donde están contenidos los números.
