Uso

Para poder hacer uso del programa es necesario tener instalado Netbeans, se debe cargar el proyecto y una vez que esté preparado en el entorno gráfico se da click en Build Project cuyo ícono es un martillo o presionar F11, una vez que se haya completado esta tarea se está listo para cliquear Run Project que está representado por un triángulo verde, o apretar F6. Esto generará la salida correspondiente en el cuadro de Output.
